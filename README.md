# vue+elementui

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

## 目录结构描述
``` bash
├── src
│   ├── api                 
│   │   └── login                // 请求文件
│   ├── assets                  // 资源文件
│   │   └── img 
│   ├── axios                   
│   │   └── index               // axios请求方法统一管理
│   ├── commone                 // 全局less配置
│   ├── component               // 公用vue组件
│   ├── config                  // 常用的配置文件
│   ├── filter                  // 全局过滤函数
│   ├── mixins                  // 全局混入和单文件混入
│   ├── router                  // 路由
│   ├── utils                   // 工具
│   │   ├── common              // 公用js方法
│   │   ├── const               // 公用常量
│   │   └── validate            // 公用校验方法
│   ├── views                   // 视图层
│   ├──store                    // vuex状态管理
│   │   └── index.js
│   │──config
│   │   └── index.js
│   │   
│   ├── App.vue
│   └── main.js
├── node_modules
└── Readme.md 
```
